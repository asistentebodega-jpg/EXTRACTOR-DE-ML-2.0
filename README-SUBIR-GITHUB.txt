INSTRUCCIONES PARA GITHUB

1. Este ZIP trae un index.html autocontenido para evitar errores 404.
2. Sube index.html a la raiz del repositorio y reemplaza el anterior.
3. Puedes subir tambien los demas archivos como respaldo, pero index.html ya incluye la app.
4. No abras app.js como pagina. La pagina principal siempre es index.html.
5. Si usas GitHub Pages: Settings > Pages > Deploy from branch > main / root.
6. Despues de subir, espera el despliegue y prueba con Ctrl + F5.
